# Sistema de Control de Acceso Basado en Roles (RBAC) con Django

## Descripción

Este proyecto implementa un sistema de **Control de Acceso Basado en Roles (RBAC)** desarrollado con Django.

## Roles Implementados

### Administrador (Admin)

- Acceso total al sistema.
- Creación de usuarios.
- Gestión de roles.

### Docente (Teacher)

- Acceso al panel docente.
- Gestión académica.

### Estudiante (Student)

- Acceso al panel estudiantil.
- Consulta de materias y tareas.

## Tecnologías Utilizadas

- Python
- Django
- SQLite
- HTML
- CSS
